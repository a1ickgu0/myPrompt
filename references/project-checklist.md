# 项目管理检查清单

基于 PMBOK 框架，用于项目审查或自查。

## 一、项目启动阶段

### 项目章程
- [ ] 项目目的和目标已明确
- [ ] 项目经理已授权
- [ ] 高层干系人已识别
- [ ] 总体里程碑已定义
- [ ] 初步预算已批准

### 干系人识别
- [ ] 干系人清单已建立
- [ ] 干系人期望已收集
- [ ] 干系人分类（权力/利益矩阵）

---

## 二、项目规划阶段

### 范围管理
- [ ] 需求已收集并记录
- [ ] 需求已确认和签字
- [ ] WBS 已创建（到工作包级别）
- [ ] 范围基准已建立

### 进度管理
- [ ] 活动已定义
- [ ] 活动依赖关系已确定
- [ ] 资源已估算
- [ ] 工期已估算
- [ ] 关键路径已识别
- [ ] 进度基准已批准

### 成本管理
- [ ] 成本已估算
- [ ] 预算已批准
- [ ] 成本基准已建立
- [ ] 应急储备已考虑

### 风险管理
- [ ] 风险已识别（风险登记册）
- [ ] 风险已定性分析
- [ ] 风险已定量分析（高优先级）
- [ ] 风险应对计划已制定
- [ ] 风险责任人已指定

### 质量管理
- [ ] 质量标准已定义
- [ ] 质量保证活动已规划
- [ ] 质量控制方法已确定

### 沟通管理
- [ ] 沟通需求已分析
- [ ] 沟通计划已制定
- [ ] 汇报机制已建立

---

## 三、项目执行阶段

### 团队管理
- [ ] 团队成员已到位
- [ ] 角色职责已明确
- [ ] 团队建设活动已开展
- [ ] 绩效反馈机制已建立

### 质量保证
- [ ] 质量审计已按计划执行
- [ ] 过程合规性已检查
- [ ] 改进建议已跟踪

### 信息分发
- [ ] 项目状态按计划报告
- [ ] 干系人沟通按计划执行
- [ ] 问题已及时上报

---

## 四、项目监控阶段

### 进度监控
- [ ] 进度偏差已分析
- [ ] 挣值分析已执行（如有需要）
- [ ] 纠偏措施已采取

### 成本监控
- [ ] 实际成本已跟踪
- [ ] 成本偏差已分析
- [ ] 预测完成成本（EAC）

### 质量控制
- [ ] 质量指标已测量
- [ ] 不合格项已记录
- [ ] 纠正措施已执行

### 风险监控
- [ ] 风险状态已更新
- [ ] 新风险已识别
- [ ] 风险应对有效性已评估

### 变更控制
- [ ] 变更请求已提交
- [ ] 变更影响已评估
- [ ] 变更已批准/拒绝
- [ ] 变更已实施并记录

---

## 五、项目收尾阶段

### 交付验收
- [ ] 交付物已完成
- [ ] 客户验收已签字
- [ ] 移交文档已完成

### 项目关闭
- [ ] 合同已关闭
- [ ] 资源已释放
- [ ] 经验教训已总结
- [ ] 项目文档已归档
