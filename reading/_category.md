# 阅读类 Skills

提供结构化的内容阅读与信息提取能力，帮助用户高效理解论文、新闻、技术文档等各类材料。

## 包含的 Skills

| Skill | 说明 | 适用场景 |
|-------|------|---------|
| [paper-analysis](paper-analysis/skill.md) | 论文阅读与应用洞察 | 学术论文解读，提取产品应用机会 |
| [news-briefing](news-briefing/skill.md) | 新闻简报生成 | 新闻报道快速提取核心事实 |
| [industry-intelligence](industry-intelligence/skill.md) | 行业技术情报顾问 | 行业报告、技术文章深度分析与咨询建议 |

## 通用特点

- 输入模式：`page`（页面内容）
- 输出语言：中文为主，专业术语保留原文
- 输出格式：结构化 Markdown
